# threat-protection-server

Server and website repository for Kuamini Systems.

## Scope

This repository now contains only server/web assets.
Agent runtime, installer artifacts, and agent-specific APIs/UI were moved to the standalone repository:

- https://github.com/vikneeswaran/threat-protection-agent

## Local Development

- Node.js 20+
- pnpm 10+

Install and run:

1. `pnpm install`
2. `pnpm dev`

## Validation

- `pnpm lint`
- `pnpm run type-check`
- `pnpm test`
